# Setup Guide – NAS Data Management Platform

## 📌 Requirements
- Node.js (v18+)
- npm
- Git

---

## 📥 Clone Project
git clone https://github.com/YOUR_USERNAME/nas-data-management-platform.git
cd nas-data-management-platform


---

## ▶️ Run Backend
cd backend
npm install
npm start


---

## ▶️ Run Frontend
cd frontend
npm install
npm run dev


---

## 🌐 Access App
http://localhost:5173


---

## 🧪 Test APIs
Use Postman:

- GET /api/files
- POST /api/chat

---

## ⚠️ Notes
- Backend must run before frontend
- Ports must not conflict
