# Frontend – NAS Data Management Platform (Phase 1)

## 📌 Overview
The frontend is built using React.  
It provides an interactive dashboard to manage files and interact with AI.

---

## 🧠 Features
- Dashboard overview
- File Explorer
- AI Assistant chat
- File details panel
- Activity log
- Confirmation modal
- Connect & Analyze screen

---

## ⚙️ Tech Stack
- React (Vite)
- Axios
- CSS (custom design)
- Lucide Icons

---

## 📂 Structure

frontend/
├── src/
│ ├── components/
│ ├── screens/
│ ├── services/
│ ├── App.jsx
│ ├── App.css


---

## 🔁 State Management
- React useState
- Centralized state in App.jsx

---

## 🔌 API Integration
- Axios used to call backend APIs:
  - /api/files
  - /api/chat

---

## 🎯 UX Design
- Clean dashboard layout
- Sidebar navigation
- Responsive UI
- Safe operations via modal confirmation

---
